# CertPass AI — 사용자 가이드
> Certificate Pass AI | 정보처리기사 실기 AI 학습 앱

---

## 🎯 CertPass AI가 뭔가요?

**Certificate Pass AI** — AI가 기출문제를 분석해서 유사 문제를 만들어주고, 틀리면 바로 해설해주는 정처기 실기 학습 앱이에요.

| 기존 방식 | CertPass AI |
|---|---|
| 기출 PDF 직접 프린트 | PDF 업로드하면 AI가 새 문제 생성 |
| 정답 맞춰보고 끝 | 틀리면 AI가 즉시 이유 설명 |
| 오답 따로 정리해야 함 | 자동으로 오답노트 저장 |

---

## 🚀 시작하기

### 1단계 — 앱 접속
- 브라우저에서 앱 URL 접속
- **Google 로그인** 클릭 → Google 계정 선택
- 로그인 없이도 문제 풀기 가능 (단, 오답노트 저장 안 됨)

### 2단계 — 공부 방법 선택

앱 상단 탭에서 원하는 방법 선택:

| 탭 | 설명 |
|---|---|
| 📚 문제풀기 | 이미 준비된 AI 생성 문제 바로 풀기 |
| ❌ 오답노트 | 내가 틀린 문제 모아보기 |
| 📎 PDF 업로드 | 내 기출 PDF 올려서 새 문제 만들기 |

---

## 📚 문제 풀기

### 과목 선택
상단 필터에서 공부할 과목 선택:
- **전체** — 모든 과목 섞어서
- **소프트웨어 설계** — UML, 디자인 패턴
- **소프트웨어 개발** — 알고리즘, 자료구조
- **데이터베이스** — SQL, 정규화, 트랜잭션
- **프로그래밍 언어** — C, Java, Python 코드 읽기
- **시스템 구축관리** — 보안, 네트워크

### 문제 풀기 순서
1. 문제 읽기
2. 모르겠으면 **💡 힌트** 버튼 클릭
3. 답변 입력 후 **제출하기** 클릭
4. 정오 확인 + **🤖 AI 해설** 읽기
5. **다음 문제 →** 클릭

### 문제 유형
| 유형 | 설명 | 예시 |
|---|---|---|
| 단답형 | 개념/용어를 직접 입력 | "ACID의 원자성을 설명하시오" |
| 빈칸채우기 | 빈칸에 알맞은 말 입력 | "SELECT __ FROM users" |
| 코드완성 | 코드 결과 또는 빈 코드 입력 | Python 코드 출력값 맞추기 |

## 📎 PDF 업로드로 새 문제 만들기

### 언제 사용하나요?
- 최신 기출문제(최근 회차)를 직접 업로드해서 유사 문제를 만들고 싶을 때
- 특정 과목 문제를 집중적으로 풀고 싶을 때

### 사용 방법
1. 큐넷(https://www.q-net.or.kr) 접속
2. 자료실 → 기출문제 → 정보처리기사 PDF 다운로드
3. CertPass AI **📎 PDF 업로드** 탭 클릭
4. "PDF 파일을 클릭해서 선택하세요" 영역 클릭 → 다운받은 PDF 선택
5. "PDF 분석 중..." → "AI가 유사 문제 생성 중..." 대기 (약 10~30초)
6. 완료 메시지 확인 후 **📚 문제풀기** 탭에서 새 문제 풀기

### 저작권 안내
> ⚠️ 업로드된 PDF는 서버에 저장되지 않습니다.
> AI가 문제 유형만 파악한 뒤 완전히 새로운 유사 문제를 생성하므로 저작권 문제가 없습니다.

## ❌ 오답노트

### 오답노트란?
문제를 틀릴 때마다 자동으로 저장되는 나만의 취약점 모음이에요.

### 사용 방법
1. **❌ 오답노트** 탭 클릭
2. 틀린 문제 목록 확인
   - 빨간 박스: 내가 쓴 틀린 답
   - 초록 박스: 정답
3. **다시 풀기 →** 클릭해서 재도전
4. 반복해서 맞출 때까지 연습

### 공부 팁
> 💡 오답노트를 매일 한 번씩 돌려보세요.
> 시험 전날에는 오답노트만 집중적으로 복습하는 게 효과적이에요!

## ❓ 자주 묻는 질문 (FAQ)

**Q. 로그인 안 해도 되나요?**
A. 문제 풀기와 AI 해설은 로그인 없이도 사용 가능해요. 단, 오답노트 저장은 Google 로그인이 필요해요.

**Q. PDF 파일 크기 제한이 있나요?**
A. 10MB 이하 PDF만 업로드 가능해요. 큐넷 기출 PDF는 대부분 이 범위 안에 있어요.

**Q. AI 해설이 틀릴 수도 있나요?**
A. AI도 가끔 실수할 수 있어요. 중요한 개념은 교재나 공식 해설과 함께 확인하세요.

**Q. 내 오답노트 데이터는 안전한가요?**
A. 본인 계정으로만 접근 가능하며, 다른 사용자에게는 절대 보이지 않아요.

**Q. 어떤 브라우저에서 사용할 수 있나요?**
A. Chrome, Safari, Edge 최신 버전을 권장해요.

**Q. 모바일에서도 사용 가능한가요?**
A. 네, 모바일 브라우저에서도 사용 가능해요.
