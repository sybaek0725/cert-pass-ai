```javascript
  alert('PDF 파일만 업로드 가능합니다.');
  return;
}

  alert('10MB 이하 파일만 업로드 가능합니다.');
  return;
}
```

## 🚨 절대 하면 안 되는 것

| 행동 | 이유 |
|---|---|
| `service_role` 키를 프론트에 넣기 | RLS 완전 우회 가능, DB 전체 노출 |
| `.env`를 GitHub에 커밋 | 키 전체 탈취 가능 |
| `sk-ant-` API 키를 클라이언트 번들에 포함 | 무제한 API 호출 가능, 과금 폭탄 |
| RLS 없이 Supabase 테이블 오픈 | 모든 유저 데이터 열람 가능 |
