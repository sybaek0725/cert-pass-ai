# CertPass AI (Certificate Pass AI) — 기술 설계서

## 1. 기술 스택

| 레이어 | 선택 | 이유 |
|---|---|---|
| Frontend | React + Vite | 빠른 세팅, 컴포넌트 재활용 |
| Styling | Tailwind CSS (CDN) | 빠른 UI 구현 |
| PDF 파싱 | pdf.js (CDN) | 브라우저 내 파싱, 서버 불필요 |
| AI | Claude API (Anthropic) | Artifact 내장 → 비용 0 |
| DB | Supabase (무료 티어) | 오답노트, 유저 진도 저장 |
| 배포 | Vercel (무료) | GitHub 연동 자동 배포 |
| 인증 | Supabase Auth | 소셜 로그인 (Google, GitHub) |

## 5. Supabase 스키마

```sql
-- 유저 (Supabase Auth 연동)
-- auth.users 테이블 자동 생성됨

-- 문제 뱅크 (AI 생성 문제 캐싱)
CREATE TABLE questions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  subject TEXT NOT NULL,          -- 과목
  type TEXT NOT NULL,             -- 단답형/빈칸/코드완성
  question TEXT NOT NULL,
  answer TEXT NOT NULL,
  hint TEXT,
  explanation TEXT,
  keywords TEXT[],
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 학습 세션
CREATE TABLE study_sessions (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  subject TEXT,
  total_count INT DEFAULT 0,
  correct_count INT DEFAULT 0,
  duration_seconds INT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 오답노트
CREATE TABLE wrong_answers (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  user_id UUID REFERENCES auth.users(id),
  question_id UUID REFERENCES questions(id),
  my_answer TEXT,
  reviewed BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- RLS (Row Level Security) 설정
ALTER TABLE study_sessions ENABLE ROW LEVEL SECURITY;
ALTER TABLE wrong_answers ENABLE ROW LEVEL SECURITY;

CREATE POLICY "유저 본인 데이터만 접근"
  ON study_sessions FOR ALL
  USING (auth.uid() = user_id);

CREATE POLICY "유저 본인 오답만 접근"
  ON wrong_answers FOR ALL
  USING (auth.uid() = user_id);
```

## 6. 핵심 로직 — PDF 파싱 플로우

```javascript
// lib/pdfParser.js
import * as pdfjsLib from 'pdfjs-dist';

export async function parsePdf(file) {
  const arrayBuffer = await file.arrayBuffer();
  const pdf = await pdfjsLib.getDocument({ data: arrayBuffer }).promise;
  
  let fullText = '';
  for (let i = 1; i <= pdf.numPages; i++) {
    const page = await pdf.getPage(i);
    const content = await page.getTextContent();
    fullText += content.items.map(item => item.str).join(' ') + '\n';
  }
  return fullText;
}
```
