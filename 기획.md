## 1. 프로덕트 개요

| 항목 | 내용 |
|---|---|
| 서비스명 | **CertPass AI** |
| 풀 네임 | Certificate Pass AI |
| 슬로건 | "기출 풀고, AI 해설 듣고, 합격한다" |
| 타겟 | 정보처리기사 실기 준비생 (IT 종사자, 취준생) |
| 핵심 가치 | 기출 PDF 업로드 → AI가 유사 문제 재생성 → 즉시 해설 |
| 배포 채널 | 네이트 앱 + 커리어리 / 에브리타임 바이럴 |

---

## 2. 문제 정의

- 정처기 실기 기출문제는 큐넷 PDF로만 공개됨 → 모바일 최적화 학습 환경 없음
- 기존 앱들은 객관식 위주 → 실기 서술형 대응 부족
- AI 해설 없음 → 틀려도 왜 틀렸는지 이해 어려움

---

## 3. 핵심 기능 (MVP)

### 3-1. PDF 업로드 & 문제 추출
- 유저가 큐넷 기출 PDF 업로드
- pdf.js로 텍스트 파싱
- Claude API가 문제 유형 분류 후 **완전히 새로운 유사 문제로 재생성** (저작권 안전)

### 3-2. 문제 풀기
- 단답형 / 빈칸채우기 / 코드 완성 유형 지원
- 타이머 모드 (실제 시험 환경 모사)
- 힌트 요청 기능

### 3-3. AI 해설
- 답 제출 후 Claude가 즉시 해설 생성
- "왜 이게 답인지" + "자주 헷갈리는 포인트" 포함
- 오답 시 관련 개념 추가 설명

### 3-4. 오답노트
- 틀린 문제 자동 저장
- 오답 패턴 분석 (어떤 과목에서 많이 틀리는지)
- 재시험 모드 (오답만 다시 풀기)

### 3-5. 공유 카드 (바이럴 장치)
- "나의 정처기 실기 준비 현황" 이미지 카드 생성
- 과목별 정답률, 총 풀이 수, 예상 합격률 시각화
- SNS 공유 버튼

## 4. 사용자 플로우 (User Flow)

### 첫 방문자
```
앱 접속
  → 로그인 (Google) 또는 둘러보기(비로그인)
  → 홈 화면: 과목 선택 or PDF 업로드 선택
```

### 문제풀기 플로우
```
과목 선택 (전체 / 소프트웨어 설계 / DB 등)
  → 문제 카드 표시
  → 답변 입력
    ├── 힌트 요청 가능 (💡 힌트 버튼)
    └── 제출
        ├── 정답 → ✅ 정답 피드백 + AI 해설
        └── 오답 → ❌ 오답 피드백 + AI 해설 + 오답노트 자동 저장
  → 다음 문제 →
```

### PDF 업로드 플로우
```
📎 PDF 업로드 탭 클릭
  → 큐넷 기출 PDF 파일 선택 (10MB 이하)
  → "PDF 분석 중..." 로딩
  → AI가 유사 문제 자동 생성
  → 문제풀기 탭으로 이동 → 새 문제 바로 풀기
```

### 오답노트 플로우
```
❌ 오답노트 탭 클릭
  → 틀린 문제 목록 확인
  → 과목별 필터링
  → 문제 클릭 → 다시 풀기
  → 공유 카드 생성 → SNS 공유
```

## 6. 앱 내 사용자 가이드 문구

### 온보딩 (첫 접속 시)
```
CertPass AI에 오신 걸 환영해요! 🎯

3가지 방법으로 공부할 수 있어요:
1️⃣ 📚 문제풀기 — AI가 만든 유사 문제를 바로 풀어요
2️⃣ 📎 PDF 업로드 — 내 기출 PDF를 올리면 AI가 새 문제 생성
3️⃣ ❌ 오답노트 — 틀린 문제만 모아서 다시 풀어요
```

### 빈 오답노트 안내
```
🎉 오답이 없어요!
문제를 풀다 보면 틀린 문제가 여기에 모입니다.
```

### PDF 업로드 안내
```
큐넷(q-net.or.kr)에서 다운받은 기출문제 PDF를 올려주세요.
AI가 저작권 안전한 유사 문제로 새롭게 만들어드려요.
PDF는 서버에 저장되지 않습니다.
```

### 힌트 안내
```
💡 힌트: [힌트 내용]
```

### AI 해설 로딩 안내
```
🤖 해설 생성 중...
```
